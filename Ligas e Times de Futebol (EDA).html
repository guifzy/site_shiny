Função para Pré-Processamento

Esta etápa visa sanar os príncipais problemas quando se trata de dados não tratados. Aqui, resolvemos problemas como:

  • Troca de NA por 0;

  • Conversão de valores para numérico;

  • Junção de tabelas;

  • Adição de colunas necessárias.

preprocess_league_data <- function(caminho, main, aux, add = NULL, league) {
 
  main_df <- read_excel(caminho, sheet = main)
  aux_df <- read_excel(caminho, sheet = aux)
  
  # converter a coluna 'pay' para numérico
  aux_df <- aux_df %>% mutate(pay = as.numeric(pay))
  main_df <- main_df %>% mutate(season_profit = as.numeric(season_profit))
  
  
  # full join com o dataframe principal
  final_df <- full_join(main_df, aux_df, by = "cod")
  
  # se tiver adicional, realizamos mais uma junção
  if (!is.null(add)) {
    add_df <- read_excel(caminho, sheet = add)
    final_df <- full_join(final_df, add_df, by = "cod")
  }
  
  # substituir valores NA por 0
  final_df[is.na(final_df)] <- 0
  
  # adicionar nome da liga
  final_df <- final_df %>% mutate(league_name = league)
  
  return(final_df)
}

Problemática

Quando falamos de futebol, por consequência estamos lidando com grandes quantias em investimentos e apostas.

Com esta análise, é possível tomar decisões difíceis em relação a investimento em times com base em características padrões de desempenho.

Analisar estes dados, significa detectar padrões em times de diferentes ligas, facilitando decisões que a um olhar descuidado, poderiam ser difíceis e arriscadas.

Conclusões

Com a visualização do lucro da temporada, stakeholders podem tomar decisões informadas sobre investimentos e estratégias financeiras.

A análise de clusters pode ajudar a identificar padrões de desempenho semelhantes entre os times, o que pode ser útil para treinadores e analistas. Vale ressaltar que a visualização dos gráficos e o modelo de predição, podem ser ajustados para visualização de ligas e times específicos.

A partir do modelo treinado, pode-se tirar conclusões sobre um possível rendimento de times em relação ao seu desempenho, como, por exemplo, um time que marca mais gols tem uma probabilidade maior de ir bem no setor financeiro.

Com melhores no modelo de predição, como a adição de um algorítimo de validação cruzada, pode tornar esta ferramenta ainda mais robusta.
